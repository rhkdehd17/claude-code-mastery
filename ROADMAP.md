# 개발자 웹이력서 개발 로드맵

## 프로젝트 개요
HTML, CSS, JavaScript, TailwindCSS를 활용한 반응형 개발자 웹이력서 개발

---

## 📋 기술 스택
- **HTML5**: 마크업 구조
- **CSS3 / TailwindCSS**: 스타일링 및 반응형 디자인
- **JavaScript**: 동적 기능 및 상호작용

---

## 🎯 Phase 1: 프로젝트 초기화 및 구조 설계

### 1.1 프로젝트 설정
- [ ] 프로젝트 폴더 구조 생성
  - `index.html` (메인 페이지)
  - `css/styles.css` (커스텀 스타일)
  - `js/script.js` (JavaScript 기능)
  - `assets/` (이미지, 아이콘 등)
- [ ] TailwindCSS 설정 및 연결
- [ ] 기본 HTML 템플릿 작성

### 1.2 레이아웃 설계
- [ ] 전체 페이지 구조 계획
- [ ] 섹션별 와이어프레임 작성
- [ ] 반응형 그리드 레이아웃 설계

## 🎨 Phase 2: UI/UX 디자인 및 레이아웃 구현

### 2.1 헤더 영역
- [ ] 네비게이션 바 제작
- [ ] 프로필 사진 및 기본 정보 배치
- [ ] 소셜 미디어 링크 추가

### 2.2 주요 섹션 구현
- [ ] **자기소개 (About)**: 간단한 소개글
- [ ] **경력 (Experience)**: 경력 기술
- [ ] **기술 스택 (Skills)**: 사용 기술 태그
- [ ] **프로젝트 (Projects)**: 프로젝트 카드
- [ ] **학력 (Education)**: 학력 정보
- [ ] **연락처 (Contact)**: 이메일, 전화, 깃허브 링크

### 2.3 스타일링
- [ ] TailwindCSS를 활용한 전체 스타일 적용
- [ ] 색상 팔레트 정의 (primary, secondary, accent)
- [ ] 타이포그래피 설정
- [ ] 버튼, 카드, 배지 등 컴포넌트 스타일링
